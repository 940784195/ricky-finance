# 界面设计指南 - 客户资产记录系统

## 一、设计灵感来源

本设计指南借鉴了 **Actual Finance** 的优秀设计模式，这是一款现代化的财务管理软件，具有简洁专业的界面风格。

---

## 二、核心设计原则

### 2.1 视觉层次
- **主内容区**：突出显示资产数据和关键指标
- **侧边导航**：提供功能入口，保持简洁
- **卡片式布局**：信息模块化，提高可读性

### 2.2 色彩系统

| 用途 | 颜色建议 | 使用场景 |
|------|---------|---------|
| **主背景** | 深色系 (#1a1a2e) | 整体界面背景 |
| **卡片背景** | 中深色 (#16213e) | 数据卡片 |
| **主文字** | 白色/浅灰 (#eaeaea) | 主要内容文字 |
| **辅助文字** | 灰色 (#8b8b8b) | 次要信息 |
| **正数/增长** | 绿色 (#4ade80) | 资产增长显示 |
| **负数/下降** | 红色 (#f87171) | 资产下降显示 |
| **中性** | 蓝色 (#60a5fa) | 常规数字 |
| **强调色** | 紫色 (#a78bfa) | 按钮、链接 |

### 2.3 交互模式

1. **悬停效果**：卡片和按钮添加微妙的悬停动画
2. **平滑过渡**：页面切换和状态变化使用过渡动画
3. **即时反馈**：操作后立即显示结果或状态提示
4. **键盘导航**：支持快捷键操作

## 三、核心组件设计

### 3.1 卡片组件 (Card)
```tsx
// 参考 Actual 的设计模式
<Card>
  <CardHeader>客户资产概览</CardHeader>
  <CardContent>
    // 资产数据展示
  </CardContent>
</Card>
```

**设计要点**：
- 圆角：6-8px
- 阴影：柔和的一层阴影
- 内边距：15px 左右
- 背景色：对比主背景略浅

### 3.2 数据展示卡片

| 组件类型 | 设计说明 |
|---------|---------|
| **资产总额卡片** | 大字体显示金额，附带增长/下降指示器 |
| **趋势卡片** | 内嵌折线图，展示变化趋势 |
| **分类卡片** | 饼图展示资产配置比例 |
| **列表卡片** | 客户列表或记录列表 |

### 3.3 按钮组件

| 按钮类型 | 样式特点 |
|---------|---------|
| **Primary** | 主色调填充，白色文字 |
| **Secondary** | 透明背景，边框高亮 |
| **Bare** | 仅文字，悬停高亮 |
| **Menu Button** | 用于菜单选择 |

### 3.4 表格组件
- 支持行悬停高亮
- 交替行背景色
- 数字右对齐
- 支持排序功能

## 五、关键界面设计

### 5.1 资产概览页

**核心元素**：
- 资产总额卡片（带增减百分比）
- 资产趋势图表（最近6个月）
- 资产分类占比
- 快速操作入口

### 5.2 客户列表页

**核心元素**：
- 搜索和筛选功能
- 客户卡片列表
- 客户资产概览（最近记录）
- 排序和分页

### 5.3 资产录入页

**核心元素**：
- 表单输入区域
- 资产类型选择
- 金额输入（带格式化）
- 日期选择器
- 备注字段

### 5.4 报表页

**核心元素**：
- 时间范围选择
- 多种图表类型切换
- 数据导出按钮
- 报表详情展示

## 六、交互细节

### 6.1 动画效果

| 场景 | 动画类型 |
|------|---------|
| 页面切换 | 淡入淡出 / 滑动 |
| 卡片加载 | 骨架屏占位 |
| 数字变化 | 计数动画 |
| 按钮点击 | 缩放反馈 |

### 6.2 响应式适配

- **桌面端**：多列网格布局
- **平板端**：自适应双列布局
- **移动端**：单列堆叠布局

### 6.3 无障碍支持

- 键盘导航支持
- 屏幕阅读器友好
- 颜色对比度达标
- 焦点状态清晰

## 七、设计资源

### 图标系统
- 建议使用 Lucide 图标库
- 统一的图标风格（线性/填充）
- 合理的图标尺寸

### 字体系统
- 主字体：Inter 或 SF Pro
- 等宽字体：用于数字显示
- 字重层级：400(常规)、500(中等)、600(半粗)、700(粗)
