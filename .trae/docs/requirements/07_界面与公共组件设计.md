#### 4.6.2 Select 下拉菜单组件

**设计规范**：
- **选项文字颜色**：黑色 (#000000)，确保高可读性
- **选项背景色**：白色 (#ffffff)，提供良好对比度
- **选中状态**：高亮显示，背景色 #f3f4f6
- **悬停状态**：背景色 #e5e7eb，文字保持黑色
- **下拉箭头**：标准下拉指示器，位置在输入框右侧

**实现方式**：
```css
/* Select dropdown styles */
.form-control option {
  color: #000000 !important;
  background: #ffffff;
}

.form-control {
  color: var(--text-primary);
}

.form-control:focus option {
  color: #000000 !important;
  background: #ffffff;
}
```

### 4.7 按钮交互与表单提交

#### 4.7.1 表单提交按钮设计

**按钮功能要求**：
- **类型**：Primary 主按钮
- **文字**：保存
- **绑定方式**：支持 form submit 事件 + click 事件双重绑定
- **防重复提交**：点击后按钮显示 loading 状态，禁用重复点击

**事件绑定方式**：
1. **form 原生提交事件**：监听 `submit` 事件，阻止默认行为
2. **按钮点击事件**：同时监听 `click` 事件，确保点击时触发
3. **调试支持**：添加 console.log 用于排查问题

**关键实现代码**：
```javascript
// 方式1：表单 submit 事件监听
document.getElementById('recordForm').addEventListener('submit', function(e) {
  console.log('表单提交事件触发');
  e.preventDefault();
  handleSubmit();
});

// 方式2：按钮 click 事件监听
document.getElementById('saveRecordBtn').addEventListener('click', function(e) {
  console.log('保存按钮点击事件触发');
  e.preventDefault();
  handleSubmit();
});
```

#### 4.7.2 退出登录功能

**功能描述**：
- **位置**：侧边栏底部
- **样式**：Bare 类型按钮，红色文字
- **交互流程**：
  1. 用户点击"退出登录"
  2. 前端清除 localStorage 中的 token 和 user 信息
  3. 调用后端登出 API（可选）
  4. 跳转到登录页面 (login.html)
  5. 清空登录表单状态

**实现代码**：
```javascript
// public/js/api.js
async function logout() {
  localStorage.removeItem('token');
  localStorage.removeItem('user');
  window.location.href = '/login.html';
}
```
