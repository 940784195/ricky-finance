## 4. 界面设计

### 4.1 报表导出页

#### 设计规范应用

- **色彩系统**: 主背景 #1a1a2e，卡片背景 #16213e
- **文字颜色**: 主文字 #eaeaea，辅助文字 #8b8b8b
- **按钮样式**: 主按钮使用强调色 #a78bfa
- **表格设计**: 交替行背景色，数字右对齐
- **进度条**: 导出进度使用蓝色 #60a5fa

#### 页面功能模块

1. **导出配置区**（卡片样式）
   - 报表类型选择：
     - 全部资产记录
     - 资产汇总表
     - 资产趋势分析报告
     - 自定义报表
   - 导出格式选择：
     - Excel (.xlsx) - 推荐
     - CSV (.csv)
     - PDF (.pdf) - 规划中
     - 格式图标展示
   - 时间范围选择：
     - 快捷选项：本月、上月、本季度、本年、全部
     - 自定义：开始日期 - 结束日期
   - 数据范围选择：
     - 全部成员
     - 指定成员（多选）
     - 全部资产类型
     - 指定资产类型（多选）
   - 报表模板选择（如适用）

2. **报表预览区**（卡片样式）
   - 预览报表结构
   - 数据量统计：
     - 记录数
     - 成员数
     - 预估文件大小
   - 数据样本展示（前5行）
   - 导出配置摘要

3. **操作区**
   - 生成预览按钮
   - 导出按钮（主按钮样式）
   - 保存配置按钮（保存为模板）

4. **历史导出记录区**（卡片样式）
   - 表格展示：
     - 报表名称
     - 导出时间
     - 格式
     - 文件大小
     - 操作：下载、删除
   - 支持分页
   - 支持搜索历史记录
   - 显示导出状态（成功/失败/进行中）

#### 交互细节

1. **导出配置交互**
   - 报表类型切换显示对应的配置选项
   - 格式选择显示对应图标
   - 时间范围快捷选项点击自动填充日期
   - 成员/类型选择支持全选/反选
   - 配置变更自动更新预览数据

2. **导出过程交互**
   - 点击导出按钮显示进度弹窗
   - 进度条实时更新
   - 显示导出步骤（准备数据、生成文件、下载）
   - 导出完成自动开始下载
   - 导出失败显示错误信息和重试按钮

3. **历史记录交互**
   - 下载按钮点击直接下载文件
   - 删除按钮点击显示确认弹窗
   - 搜索框支持按报表名称、时间筛选
   - 表格支持排序

4. **配置模板交互**
   - 保存配置按钮打开弹窗输入模板名称
   - 模板列表支持加载、删除
   - 快速应用常用配置

### 4.2 系统设置页

#### 设计规范应用

- **色彩系统**: 主背景 #1a1a2e，卡片背景 #16213e
- **导航菜单**: 激活菜单项使用强调色 #a78bfa
- **表单设计**: 统一的输入框样式，验证提示
- **安全提示**: 重要操作显示红色警告

#### 页面功能模块

1. **设置导航菜单**（左侧垂直导航）
   - 户主菜单：
     - 个人信息
     - 家庭设置
     - 资产类型（快捷入口）
     - 数据管理
   - 系统管理员菜单（额外）：
     - 系统配置
     - 用户管理
     - 系统日志

2. **设置内容区**（对应菜单项）
   - **个人信息设置**：
     - 头像设置（上传/预览）
     - 姓名输入
     - 邮箱输入
     - 修改密码表单：
       - 当前密码
       - 新密码
       - 确认密码
     - 保存按钮
   - **家庭设置**：
     - 家庭名称输入
     - 创建时间显示（只读）
     - 户主信息显示
     - 转移户主权限按钮（规划中）
     - 保存按钮
   - **数据管理**：
     - 数据备份：
       - 备份按钮
       - 备份历史列表
       - 下载备份文件
     - 数据恢复（规划中）
     - 数据清空（危险操作，需确认）
   - **系统配置**（管理员）：
     - 系统参数表单
     - 保存配置按钮
   - **系统日志**（管理员）：
     - 日志列表表格
     - 日志级别筛选
     - 时间范围筛选
     - 导出日志按钮

#### 交互细节

1. **设置菜单交互**
   - 点击菜单项切换内容区
   - 激活菜单项高亮显示（紫色）
   - 菜单支持折叠/展开

2. **表单交互**
   - 输入框聚焦边框高亮
   - 实时验证输入格式
   - 错误提示红色显示
   - 保存按钮点击显示加载状态
   - 保存成功显示提示消息

3. **密码修改交互**
   - 密码输入框支持显示/隐藏
   - 新密码强度提示（弱/中/强）
   - 确认密码实时比对
   - 修改成功后需重新登录

4. **数据管理交互**
   - 备份按钮点击显示进度
   - 备份完成提供下载
   - 删除备份显示确认弹窗
   - 数据清空操作：
     - 需输入确认文字
     - 二次确认弹窗
     - 红色警告提示

5. **系统日志交互**
   - 日志级别筛选（全部/信息/警告/错误）
   - 支持导出日志为文件
   - 点击日志项查看详情

6. **导航一致性**
   - 所有设置页面保持相同的布局结构
   - 侧边导航高亮当前页面
   - 支持键盘导航
