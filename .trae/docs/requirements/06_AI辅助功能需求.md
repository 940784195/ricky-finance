## 2. 数据分析代理

### 2.1 功能描述

分析客户资产变化趋势，生成智能洞察

### 2.2 能力

- 趋势分析:识别资产增长/下降趋势
- 对比分析:跨客户、跨时间的资产对比
- 异常识别:检测资产数据的异常波动
- 预测分析:基于历史数据预测未来趋势
- 洞察生成:自动生成业务洞察报告

---

## 3. 异常检测代理

### 3.1 功能描述

监控资产数据，识别异常变化并发出警报

### 3.2 能力

- 阈值监控:检测超过预设阈值的变化
- 模式识别:识别异常的数据模式
- 趋势异常:检测与历史趋势不符的变化
- 预警机制:自动触发异常预警

---

## 4. 数据同步代理

### 4.1 功能描述

确保桌面端与移动端数据一致性

### 4.2 能力

- 实时同步:检测变更并立即同步
- 冲突解决:处理数据同步冲突
- 增量同步:仅同步变化的数据
- 离线支持:支持离线操作后的同步

---

## 5. 提醒通知代理

### 5.1 功能描述

提醒客户/管理员定期更新资产数据

### 5.2 能力

- 定期提醒:按计划发送数据更新提醒
- 逾期提醒:提醒未及时更新的用户
- 事件通知:资产变化达到阈值时的通知
- 多渠道推送:支持邮件、短信、APP推送

---

## 6. 报告生成代理

### 6.1 功能描述

自动生成格式化的资产报告

### 6.2 能力

- 定期报告:按周/月/季度生成报告
- 自定义报告:根据需求生成特定报告
- 多格式输出:支持PDF、Excel、HTML
- 模板支持:支持多种报告模板

---

## 7. 界面设计

### 7.1 AI助手面板设计

#### 设计规范应用

- **色彩系统**: 主背景 #1a1a2e，卡片背景 #16213e
- **强调色**: AI相关元素使用紫色 #a78bfa
- **图标风格**: 使用统一的线性图标
- **动画效果**: 平滑过渡，加载动画

#### 面板功能模块

1. **AI助手入口**
   - 页面右下角悬浮按钮
   - 显示AI图标和"智能助手"提示
   - 新消息时有红点提示

2. **助手面板**（点击展开）
   - 面板标题栏：
     - AI助手标题
     - 最小化/关闭按钮
   - 对话区域：
     - 历史对话列表
     - AI消息气泡（紫色背景）
     - 用户消息气泡（蓝色背景）
     - 时间戳显示
   - 输入区域：
     - 文本输入框
     - 快捷问题按钮（预设问题）
     - 发送按钮
   - 功能标签页：
     - 对话
     - 洞察
     - 建议

#### 面板布局

```
┌─────────────────────────────────────┐
│  🤖 AI助手            [─] [×]  │
├─────────────────────────────────────┤
│                                     │
│  [AI] 你好！我是你的智能资产    │
│        助手，有什么可以帮你的？     │
│  [10:30]                          │
│                                     │
│  [用户] 分析一下本月资产情况     │
│  [10:32]                          │
│                                     │
│  [AI] 好的，本月你的家庭总资   │
│        产增长了5.2%，主要贡献者是  │
│        股票类资产...               │
│  [10:32]                          │
│                                     │
├─────────────────────────────────────┤
│  [📊趋势] [💡建议] [📈预测]  │
│  ┌─────────────────────────────┐  │
│  │ 输入你的问题...            │  │
│  └─────────────────────────────┘  │
│                    [发送]         │
└─────────────────────────────────────┘
```

#### 交互细节

1. **面板交互**
   - 点击悬浮按钮展开/收起面板
   - 面板支持拖拽移动位置
   - 新消息自动滚动到底部
   - 支持键盘快捷键（Ctrl+K打开）

2. **对话交互**
   - 快捷问题按钮点击自动发送
   - AI回复有打字动画效果
   - 支持复制AI回复内容
   - 支持清空对话历史

3. **AI响应**
   - 显示加载动画
   - 支持Markdown格式渲染
   - 数据支持点击跳转到对应页面
   - 图表支持在对话中展示

### 7.2 智能洞察展示

#### 设计规范应用

- **色彩系统**: 主背景 #1a1a2e，卡片背景 #16213e
- **洞察分类颜色**:
  - 正向洞察：绿色 #4ade80
  - 中性洞察：蓝色 #60a5fa
  - 警告洞察：黄色 #f59e0b
  - 风险洞察：红色 #f87171

#### 洞察展示模块

1. **洞察卡片**
   - 洞察图标（根据类型）
   - 洞察标题
   - 洞察详细描述
   - 相关数据指标
   - 操作建议（如适用）
   - 时间戳
   - 操作按钮：查看详情、忽略、标记已读

2. **洞察筛选与排序**
   - 按类型筛选（全部/正向/中性/警告/风险）
   - 按时间排序（最新/最早）
   - 按重要性排序
   - 搜索功能

3. **洞察详情弹窗**
   - 完整洞察描述
   - 数据可视化图表
   - 历史对比
   - 相关记录列表
   - 操作建议
   - 分享/导出按钮

#### 交互细节

1. **洞察卡片交互**
   - 悬停阴影加深

   - 忽略按钮将洞察移至归档
   - 查看详情直接跳转相关页面

2. **筛选与搜索**
   - 筛选器实时生效
   - 搜索支持关键词匹配
   - 支持批量操作

3. **洞察详情交互**
   - 图表支持交互
   - 支持分享洞察
   - 支持导出洞察报告
   - 支持标记重要洞察

### 7.3 通知中心设计

#### 设计规范应用

- **色彩系统**: 主背景 #1a1a2e，卡片背景 #16213e
- **通知类型颜色**:
  - 信息通知：蓝色 #60a5fa
  - 成功通知：绿色 #4ade80
  - 警告通知：黄色 #f59e0b
  - 错误通知：红色 #f87171
- **未读标识**: 使用强调色 #a78bfa 圆点

#### 通知中心模块

1. **通知入口**
   - 顶部导航栏铃铛图标
   - 未读数量红点提示
   - 点击展开通知下拉面板

2. **通知列表**
   - 通知图标（根据类型）
   - 通知标题
   - 通知摘要
   - 时间显示（相对时间：刚刚/5分钟前/...）
   - 未读标识（圆点）
   - 操作按钮：标记已读、删除

3. **通知筛选**
   - 全部通知
   - 未读通知
   - 按类型筛选（信息/成功/警告/错误）

4. **通知设置**
   - 通知开关
   - 推送渠道设置
   - 静音时间段设置

#### 通知中心布局

```
┌─────────────────────────────────┐
│  🔔 通知中心        [设置]  │
├─────────────────────────────────┤
│  [全部] [未读] [类型▼]       │
├─────────────────────────────────┤
│  ● 🔔 [紫色]                │
│     新的智能洞察已生成        │
│     查看本月资产分析报告      │
│     2分钟前                │
│                                │
│  ● ⚠️ [黄色]                │
│     资产异常变化检测        │
│     张三的股票资产下降8%      │
│     15分钟前               │
│                                │
│  ● ✅ [绿色]                │
│     数据同步成功            │
│     最新资产数据已同步      │
│     1小时前                │
│                                │
└─────────────────────────────────┘
```

#### 交互细节

1. **通知面板交互**
   - 点击铃铛图标展开/收起
   - 面板支持滚动查看更多
   - 点击通知项打开详情或跳转
   - 标记全部已读按钮

2. **通知操作**
   - 悬停显示操作按钮
   - 点击标记已读移除圆点
   - 删除通知显示确认弹窗

3. **通知设置交互**
   - 开关按钮切换状态
   - 推送渠道支持多选
   - 静音时间支持自定义时间段
   - 设置自动保存
