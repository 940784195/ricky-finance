## 2. 数据字段定义

### 2.1 家庭信息字段

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | 数字 | 是 | 家庭唯一标识 |
| familyName | 字符串 | 是 | 家庭名称 |
| headId | 数字 | 是 | 户主ID |
| createdAt | 日期时间 | 是 | 创建时间 |

### 2.2 家庭成员信息字段

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | 数字 | 是 | 成员唯一标识 |
| familyId | 数字 | 是 | 所属家庭ID |
| name | 字符串 | 是 | 成员姓名 |
| shortName | 字符串 | 否 | 成员简称(用于头像显示) |
| role | 字符串 | 是 | 角色(head/member) |
| createdAt | 日期时间 | 是 | 创建时间 |

## 3. 家庭管理功能

### 3.1 户主权限功能

- **新增家庭成员**: 录入新家庭成员的基本信息
- **编辑家庭成员**: 修改家庭成员信息
- **删除家庭成员**: 删除家庭成员(级联处理关联的资产记录)
- **查看家庭成员详情**: 查看成员详细信息及资产记录
- **家庭成员列表展示**: 以卡片形式展示所有家庭成员
- **家庭统计**: 家庭成员数、家庭总资产等

### 3.2 系统管理员权限功能

- **查看所有家庭列表**: 查看系统中所有家庭信息
- **查看家庭详情**: 查看指定家庭的完整信息和资产数据
- **家庭统计**: 家庭总数、系统总资产等

## 4. 界面设计

### 4.1 家庭成员管理页(户主)

#### 设计规范应用

- **色彩系统**: 主背景 #1a1a2e，卡片背景 #16213e
- **文字颜色**: 主文字 #eaeaea，辅助文字 #8b8b8b
- **统计数字**: 正数/增长 #4ade80，中性 #60a5fa
- **按钮样式**: 主按钮使用强调色 #a78bfa
- **卡片设计**: 圆角 6-8px，柔和阴影，内边距 15px

#### 页面功能模块

1. **家庭成员统计卡片**
   - 成员总数（卡片1：显示数字 + 图标）
   - 有资产记录的成员数（卡片2：显示数字 + 占比）
   - 无资产记录的成员数（卡片3：显示数字 + 提示）
   - 每个统计卡片使用中深色背景，数字使用大号字体

2. **成员卡片网格**
   - 每个成员卡片包含：
     - 头像（圆形，使用姓名首字母或自定义头像）
     - 姓名（主文字）
     - 角色标识（标签样式：户主显示紫色标签，成员显示灰色标签）
     - 资产总额（大号字体，正数显示绿色）
     - 记录数（辅助文字）
   - 卡片操作：
     - 查看详情（悬停显示或卡片点击）
     - 编辑（图标按钮）
     - 删除（图标按钮，需二次确认弹窗）
   - 响应式网格布局：桌面端4列，平板端2-3列，移动端1-2列

3. **新增成员按钮**（页面右上角）
   - 使用主按钮样式（强调色 #a78bfa）
   - 图标 + 文字：+ 新增成员
   - 点击打开新增成员弹窗

4. **筛选与搜索**
   - 按角色筛选：下拉选择（全部/户主/成员）
   - 按姓名搜索：输入框，支持实时搜索
   - 筛选栏使用卡片样式

#### 交互细节

1. **成员卡片交互**
   - 悬停时卡片阴影加深，有轻微上浮效果
   - 点击卡片查看成员详情弹窗
   - 编辑/删除按钮悬停高亮
   - 删除操作显示确认弹窗，内容包含："确定要删除成员 [姓名] 吗？此操作将同时删除该成员的所有资产记录。"

2. **新增/编辑成员弹窗**
   - 使用设计指南中的弹窗规范
   - 表单字段：姓名、简称（用于头像）、角色选择
   - 确认/取消按钮
   - 表单验证：姓名必填

3. **筛选搜索交互**
   - 搜索输入框实时过滤结果
   - 筛选器切换立即生效
   - 无结果时显示空状态提示

### 4.2 家庭管理页(系统管理员)

#### 设计规范应用

- **色彩系统**: 主背景 #1a1a2e，卡片背景 #16213e
- **表格设计**: 行悬停高亮，交替行背景色，数字右对齐
- **统计卡片**: 使用强调色突出关键指标

#### 页面功能模块

1. **系统统计区**
   - 家庭总数（卡片1：数字 + 趋势箭头）
   - 成员总数（卡片2：数字 + 趋势箭头）
   - 系统总资产（卡片3：大号数字，绿色显示）
   - 本月活跃家庭数（卡片4：数字 + 占比）
   - 统计卡片横向排列，响应式布局

2. **家庭列表区**
   - 表格形式展示：
     - 家庭名称（左对齐，可点击查看详情）
     - 户主姓名（左对齐）
     - 成员数（右对齐）
     - 总资产（右对齐，绿色显示）
     - 创建时间（左对齐）
     - 操作列：查看详情按钮
   - 表格支持排序（点击列标题）
   - 支持分页

3. **筛选与搜索**
   - 按创建时间范围筛选：日期选择器
   - 按家庭名称搜索：输入框
   - 筛选栏使用卡片样式

#### 交互细节

1. **表格交互**
   - 行悬停时背景色略微变亮
   - 点击列标题排序（显示排序方向图标）
   - 总资产数值右对齐，正数显示绿色
   - 家庭名称可点击，打开家庭详情弹窗

2. **筛选搜索交互**
   - 日期范围选择器支持快捷选项：本月、上月、本季度、本年
   - 搜索支持模糊匹配
   - 筛选后更新统计数据

3. **家庭详情弹窗**
   - 显示家庭基本信息
   - 显示家庭成员列表
   - 显示家庭资产概览
   - 支持查看该家庭的详细资产记录
