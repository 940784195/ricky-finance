## 1. 项目概述

### 1.1 项目名称
资产管家 (Ricky Finance)

### 1.2 项目定位与目标
**软件定位**: 面向家庭用户的资产管理系统

**项目目标**: 构建一个家庭资产管理系统，用于记录、管理、分析家庭成员的各类资产，提供数据可视化、报表导出等功能。支持户主管理家庭成员，管理员查看所有家庭数据。

### 1.3 技术栈
- 后端: Node.js + Express
- 数据库: SQLite (better-sqlite3)
- 前端: HTML + CSS + JavaScript
- 图表库: Chart.js
- 测试: Jest + Playwright
- AI助手: TRAE Agents/Skills/Tools
