# Cross-Platform Sync MCP

## Overview
该MCP提供跨平台数据同步能力。

## Capabilities
- 实时同步：WebSocket实时数据推送
- 增量同步：仅同步变化数据
- 冲突检测：检测数据同步冲突
- 状态管理：管理同步状态

## Supported Platforms
- 桌面端：Windows, macOS, Linux
- 移动端：iOS, Android

## Usage
确保桌面端和移动端数据保持一致。
