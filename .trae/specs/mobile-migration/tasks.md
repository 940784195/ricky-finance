### [ ] Task P1-2: 编写 PG 建表迁移脚本
- **Priority**: P0
- **Depends On**: [P1-1]
- **Description**: 
  - 创建 `backend/db/migrate.sql`，将 5 张表 DDL 从 SQLite 转换为 PostgreSQL：
    - `INTEGER PRIMARY KEY AUTOINCREMENT` → `SERIAL PRIMARY KEY` 或 `INTEGER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY`
    - `datetime('now','localtime')` → `NOW()`
    - `CHECK(role IN ('head','member'))` → 保持（PG 支持）
    - `REFERENCES ... ON DELETE CASCADE` → 保持
    - 添加 `sync_status` 字段到 records 表（`VARCHAR(20) DEFAULT 'synced'`）
    - 添加 `local_updated_at` 和 `server_updated_at` 字段到 records 表
  - 创建 `backend/db/migrate.js` 脚本执行迁移
- **Acceptance Criteria Addressed**: [AC-1]
- **Test Requirements**:

  - `programmatic` TR-P1-2.2: 所有表正确创建，外键约束生效
- **Notes**: 参考现有 [db.js](file:///d:/TRAEworkspace/Ricky_finance_repository/Ricky_finance/server/db.js) 的 `initTables()` 函数
