# Report Generator Agent

## Overview
该智能体负责自动生成格式化的客户资产报告。

## Capabilities
- 定期报告：按周/月/季度生成报告
- 自定义报告：根据需求生成特定报告
- 多格式输出：支持PDF、Excel、HTML
- 模板支持：支持多种报告模板

## Skills Used
- data-export
- chart-visualization

## Usage
根据预设计划或用户请求，自动生成并导出报告。
