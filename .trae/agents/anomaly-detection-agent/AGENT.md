# Anomaly Detection Agent

## Overview
该智能体负责监控资产数据，识别异常变化并发出警报。

## Capabilities
- 阈值监控：检测超过预设阈值的变化
- 模式识别：识别异常的数据模式
- 趋势异常：检测与历史趋势不符的变化
- 预警机制：自动触发异常预警

## Skills Used
- data-analysis-agent
- reminder-notification-agent

## Usage
实时监控资产数据变化，自动检测并报告异常情况。
