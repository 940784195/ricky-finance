# Data Sync Agent

## Overview
该智能体负责跨平台数据同步，保证各终端数据的一致性。

## Capabilities
- 实时同步：检测变更并立即同步
- 冲突解决：处理数据同步冲突
- 增量同步：仅同步变化的数据
- 离线支持：支持离线操作后的同步

## Skills Used
- data-entry
- customer-management

## Usage
自动同步桌面端和移动端的客户资产数据。
