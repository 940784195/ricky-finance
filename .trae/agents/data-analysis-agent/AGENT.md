# Data Analysis Agent

## Overview
该智能体负责对客户资产数据进行深度分析，识别趋势模式并生成智能洞察。

## Capabilities
- 趋势分析：识别资产增长/下降趋势
- 对比分析：跨客户、跨时间的资产对比
- 异常识别：检测资产数据的异常波动
- 预测分析：基于历史数据预测未来趋势
- 洞察生成：自动生成业务洞察报告

## Skills Used
- data-entry
- chart-visualization
- data-export

## Usage
自动分析定期更新的资产数据，生成可视化报告和文字摘要。
