# Data Entry Skill

## Overview
This skill provides functionality for entering customer asset data into the system.

## Features
- Manual data input for single customer records
- Batch import from Excel/CSV files
- Data validation and format checking
- Support for multiple asset types (stocks, bonds, real estate, etc.)
- Historical data backfilling

## Usage Scenarios
- Adding new customer asset records
- Updating existing asset data periodically
- Importing bulk data from legacy systems
- Correcting or supplementing incomplete records
