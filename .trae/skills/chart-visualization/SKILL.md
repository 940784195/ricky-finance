# Chart Visualization Skill

## Overview
This skill transforms raw asset data into visual charts for better understanding.

## Features
- Line charts for tracking asset trends over time
- Bar charts for comparing assets across customers
- Pie charts for asset allocation breakdown
- Support for multiple time periods (daily, monthly, yearly)
- Interactive chart elements with tooltips
- Export charts as image files (PNG, JPEG)

## Usage Scenarios
- Displaying customer asset growth trends
- Visualizing portfolio allocation
- Comparing asset levels across different time points
- Presenting data in reports or presentations
